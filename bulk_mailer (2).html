Configuration EmailJS
EmailJS permet d'envoyer des emails directement depuis ce fichier HTML, sans serveur PHP.
Gratuit jusqu'à 200 emails/mois. Inscription sur emailjs.com
Étape 1 — Créer un compte :
Va sur emailjs.com → Sign Up (gratuit)
Étape 2 — Ajouter un service email :
Dashboard → Email Services → Add New Service → choisis Gmail ou autre → connecte ton compte → copie le Service ID
Étape 3 — Créer un template :
Dashboard → Email Templates → Create New Template
Dans le template, utilise exactement ces variables :
To: {{to_email}} | Subject: {{subject}} | Body: {{message}} | From: {{from_name}}
Sauvegarde → copie le Template ID
Étape 4 — Récupérer la clé publique :
Dashboard → Account → API Keys → copie la Public Key
✓ Configuration sauvegardée ! Tu peux maintenant envoyer de vrais emails.
